An SQL "Mod" to implement random enchantments to all rare and epic weapons and armor.

Requirements:
Azerothcore


HOW TO INSTALL:

Run the SQL query in your acore_world database to populate it with items
Place the MPQ file in the Data folder of your WoW CLIENT
Place the DBC file in the dbc folder of your Azerothcore SERVER, replace existing file

Feel free to share and modify these files whichever way you see fit.
